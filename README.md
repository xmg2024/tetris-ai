# tetris-ai

### 功能总结
1. 精美边框与UI
   - 游戏区域现在拥有漂亮的深色边框
   - 每个格子都有背景图案，视觉效果更清晰
   - 整体界面更加现代化

2. 分数系统
   - 实时显示当前分数
   - 显示历史最高分（金色高亮）
   - 最高分自动保存到本地 high_score.txt 文件

3. 双模式支持
   - 玩家模式 ：你可以自己玩，支持完整的操作
     - 方向键 Left/Right/Down ：移动和加速下落
     - Up ：旋转方块
     - Space ：直接落下（硬降）
   - AI模式 ：由强大的AI自动玩
   - 可以在游戏中随时点击按钮切换

4. 重置功能
   - 新增"Restart"按钮，随时重新开始

### 如何运行
在终端中执行：
```
python tetris_ai.py
```

游戏已经在后台启动啦！祝你玩得开心！🎮
